# Electricity Bill Calculator

## Description
This is a simple Python project that calculates the electricity bill based on the number of units consumed.

## Features
- Calculates electricity bill using slab rates.
- Accepts customer name and units consumed.
- Displays the total bill including 5% tax.

## Technologies Used
- Python

## Slab Rates

| Units | Rate |
|--------|------|
| 0 - 100 | ₹1.50/unit |
| 101 - 200 | ₹2.50/unit |
| 201 - 300 | ₹4.00/unit |
| Above 300 | ₹6.00/unit |

## How to Run

1. Install Python.
2. Run:

```bash
python main.py
```
